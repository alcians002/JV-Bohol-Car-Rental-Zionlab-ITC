# JV Bohol Car Rental

A web-based car rental platform for JV Bohol Car Rental. It allows customers to browse available vehicles, view details, and inquire about rentals for their Bohol trip. It also includes an administrative backend for managing the vehicle fleet, bookings, and website content.

## Features
*   **Public Website**: Showcases the vehicle fleet, organized by categories (Hatchback, Sedan, MPV, Pickup Truck, Van).
*   **Mobile Responsive**: Optimized design for mobile, tablet, and desktop devices.
*   **Dark Mode**: Built-in toggle for light and dark themes.
*   **Admin Dashboard**: Secure backend to manage available vehicles, bookings, and track analytics.

## Tech Stack
*   **Frontend**: HTML, vanilla CSS, JavaScript, Bootstrap 5.3
*   **Backend**: PHP (PDO for database interactions)
*   **Database**: MySQL
